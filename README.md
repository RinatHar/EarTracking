## Запуск:
Создание окружения:
`pip -m venv venv`
Активация окружения:
`venv/Scripts/activate`
Скачивание зависимостей:
`pip install -r requirements.txt`
Запуск файла для наложения сережки в реальном времени через камеру:
`python main.py`
